# Security policy

Report suspected vulnerabilities privately through GitHub Security Advisories. Do not open public issues containing credentials, private medical information, or patient data.

MediLife is a research and product prototype, not a clinical decision system. Deployments must provide authentication, authorization, transport encryption, audit logging, secret management, user-data isolation, retention controls, and an appropriate human-review process. Never submit identifiable patient data to an external model or memory provider without a lawful basis and explicit organizational approval.
